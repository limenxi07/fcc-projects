Variables

Assign variables using "=". If the variable is not defined, trying to use it will give you an error (NameError).

animal = "dog"

In the above example, the animal variable is set to a string, which can be enclosed in single quotes or double quotes. Strings can be concatenated with the + operator, and repeated with *.

>>>'ba' + 2 * 'na'
banana

The build-in function len() returns the length of a string.

>>> s = 'supercalifragilisticexpialidocious'
>>> len(s)
34

For numbers, expression syntax is straightforward: the operators +, -, * and / work just like in most other languages; parentheses () can be used for grouping.

>>> (50 - 5*6) / 4 # division always returns a floating point number.
5.0

Integer numbers (e.g. 2, 4, 20) have type int, while the ones with a fractional part (e.g. 5.0, 1.6) have type float.

To do floor division and get an integer result, use the // operator; use % to calculate the remainder.

>>> 17 // 3 # floor division discards the fractional part
5
>>> 17 % 3 # the % operator returns the remainder of the division
2

Lists & Indexing

Python knows a number of compound data types, used to group together other values. The most versatile is the list, which can be written as a list of comma-separated values (items) between square brackets. Lists might contain items of different types, but usually the items all have the same type.

>>> squares = [1, 4, 9, 16, 25]
>>> squares
[1, 4, 9, 16, 25]

The built-in function len() also applies to lists.

>>> letters = ['a', 'b', 'c', 'd']
>>> len(letters)
4

Lists and strings can be indexed. The first character has index 0. Indices may also be negative numbers, to start counting from the right.

>>> word = 'Python'
>>> word[0] # character in position 0
'P'
>>> word[5] # character in position 5
'n'
>>> word[-2] # second-last character
'o'

Lists can strings can also be sliced to obtain sublists and substrings respectively. If omitted, the first index defaults to zero, and the omitted second index defaults to the size of the list/string being sliced.

Note that the start is always included, and the end is always excluded. Attempting to use an index that is too large will result in an error (IndexError).

>>> word[0:2] # characters from position 0 (included) to 2 (excluded)
'Py'
>>> word[:2] # character from the beginning to position 2 (excluded)
'Py'
>>> word[4:] # characters from position 4 (included) to the end
'on'

Lists are mutable; you can change their content by assigning list indices or sublists to different values.

>>> cubes = [1, 8, 27, 65, 125]
>>> cubes[3] = 64 # replace the wrong value
>>> cubes
[1, 8, 27, 64, 125]

Python strings cannot be changed; they are immutable. Assigning to an indexed position in the string results in an error (TypeError). Change the contents of a string by creating a new one, then reassigning the variable to this new string.

>>> word = 'Python'
>>> word = 'J' + word[1:]
'Jython'
