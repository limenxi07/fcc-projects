Variables

Assign variables using "=", like so:

animal = "dog"

In the above example, the animal variable is set to a string, which can be enclosed in single quotes or double quotes. Strings can be concatenated with the + operator, and repeated with *. For example, the following line of code produces 'banana':

'ba' + 2 * 'na'
